# AI 체험 웹앱 - 얼굴 이미지 분석

웹캠으로 얼굴 이미지를 입력받아 재미있는 체험을 제공하는 AI 웹앱입니다.

## 주요 기능

### 1. 닮은 연예인 찾기 모드
- 얼굴 이미지의 분위기를 분석하여 유사한 이미지 분위기의 연예인을 찾아줍니다
- 이미지 분위기 유사도를 참고용으로 제공합니다

### 2. 미래 직업 추천 모드
- 얼굴 이미지 분위기 기반으로 직업 분야 예시를 제시합니다
- 참고용 정보이며, 실제 직업 선택은 관심과 노력에 따라 달라집니다

## 중요 안내

- **모든 결과는 참고용이며 재미와 체험 목적입니다**
- 외모 평가나 비교를 하지 않습니다
- AI는 얼굴만 보고 적성이나 능력을 알 수 없습니다
- 결과는 데이터에 따라 달라질 수 있습니다

## 설치 및 실행

### 필요한 환경
- Node.js 16 이상
- npm 또는 yarn

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속하세요.

### 빌드
```bash
npm run build
```

## 기술 스택

- React 18
- Vite
- TensorFlow.js (얼굴 특징 분석)

## 사용 방법

1. 웹앱에 접속합니다
2. 원하는 모드를 선택합니다 (닮은 연예인 찾기 / 미래 직업 추천)
3. 카메라 권한을 허용합니다
4. 얼굴을 가이드 안에 맞춥니다
5. 촬영 버튼을 눌러 사진을 찍습니다
6. 분석 결과를 확인합니다

## 주의사항

- 카메라 권한이 필요합니다
- HTTPS 환경에서 실행하는 것을 권장합니다 (로컬 개발 환경에서는 localhost에서도 동작합니다)
- 모바일 브라우저에서도 사용 가능합니다
